## 10. PRE-SEED FUNDING ANNOUNCEMENT

### AnnouncementBar Component (`src/components/AnnouncementBar.tsx`)
- Fixed gradient bar at the very top of the page (`z-[60]`), 40px height
- Purple gradient background: `indigo-500 → violet-500 → violet-400` at 90% opacity with backdrop blur
- Bilingual text via `t("announcement.bar")`, links to `/news` (language-aware)
- Dismissible via close button; state persisted in `sessionStorage` (`unchain-announcement-dismissed`)
- Sets CSS variable `--announcement-height` to `40px` (visible) or `0px` (dismissed) to control layout offset
